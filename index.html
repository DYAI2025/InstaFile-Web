Workflow

So schnell wird Markierung zur Datei

FlashDoc hängt sich an deine bestehende Arbeitsweise – egal ob du in der API-Doku, im Log-Viewer oder im Chat arbeitest.

1
Text markieren

Du markierst beliebigen Text im Browser – Code-Snippet, JSON-Response, Notizblock, Versandadresse.

2
FlashDoc auslösen

Wähle Kontextmenü, Shortcut oder Floating Button. Im Smart-Modus erkennt FlashDoc automatisch das passende Format.

Kontextmenü Ctrl/Cmd + Shift + S Floating Button
3
Datei fertig im Ordner

Sekunden später liegt die Datei im konfigurierten Download-Ordner – mit konsistentem Namen und optional nach Typ sortiert.

InstantFiles/json/… instant_YYYY-MM-DD_HH-MM-SS.ext
Formate & Trigger

Alles, was du im Browser siehst, wird speicherbar.

Von JavaScript über YAML bis hin zu Versandlabels: FlashDoc erkennt typische Muster und speichert im passenden Format.

Daten & APIs

Analysiere API-Responses und Tabellen später weiter – ohne Copy & Paste.

UI & Datenschutz

Sieht gut aus. Bleibt lokal.

Alle Erkennungen, Dateinamen und Statistiken laufen ausschließlich in deinem Browser. Keine Server, keine Log-Collection.

  • Verarbeitung von Inhalt ausschließlich lokal im Browser
  • Keine Netzwerk-Requests an externe Services
  • Einstellungen via chrome.storage.sync, Statistiken nur lokal
  • Manifest V3, keine inline-eval oder Remote Scripts
